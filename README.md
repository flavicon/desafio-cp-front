# Projeto Bookstore :books:

<hr>

### Bookstore é um e-commerce de livros, na aplicação é possível listar os títulos por categoria e adicionar as seleções no carinho.

### No projeto foram utilizadas as tecnologias ReactJS, Styled-components, Material-UI, Axios e Json-server.

### Para executar o projeto na sua máquina é bem simples. 

##### PS.: É necessário que esteja instalado na sua máquina o node.js na sua versão LTS e o GIT para clonar o repositório. 

<hr>

#### 1. Realize o clone do repositório.

`git clone https://github.com/flavicon/desafio-cp-front.git`

#### 2. Acesse o diretório que foi clonado.

`cd desafio-cp-front`

#### 3. Digite o comando para baixar todas as dependências.
`yarn`

#### 4. Para incializar o json-server execute o comando:
`yarn server`

#### 5. Para executar o projeto digite.
`yarn start`

### >>> O projeto irá excutar no endereço: http://localhost:3000
